# Web Dev Journey

This project contains a small web application and supporting backend code.

## Repository structure

- `public/` – static site files (HTML, CSS, JS and images)
- `backend/` – Node.js backend service
- `docs/` – planning documents
- `misc/`, `src/`, `test/`, `tools/` – additional resources

Open the site by opening any HTML file inside `public/` in your browser.
